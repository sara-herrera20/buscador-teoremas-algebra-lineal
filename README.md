# Buscador de Teoremas — Álgebra Lineal

Página web estática (solo `index.html`) con un buscador de teoremas.

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Busca por palabra clave o filtra por parcial/tema.

## Publicar en GitHub Pages
En GitHub:
- Settings → Pages
- Source: Deploy from a branch
- Branch: `main` / folder: `/ (root)`
